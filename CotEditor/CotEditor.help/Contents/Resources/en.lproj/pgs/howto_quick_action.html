Perform action by typing in CotEditor

With the Quick Action feature, you can perform menu commands just by typing few keys on your keyboard.

To perform a quick action:

  1. Launch the command bar either by choosing View > Show Quick Actions or by pressing the Command-K keys.
  2. Type characters into the command bar. The matched actions will be shown incrementally below the field.
    • It matches the menu commands having all the typed characters in the order in the intermediate submenus and its title.
  3. Select the action to perform in the list by pressing the ↓ or ↑ key.
  4. Press the Return key to perform the selected action.

To close the command bar without taking any action:

  • Press the Esc key, or Command-. keys.
  • Press the Return key while no action candidates are shown.
  • Click somewhere outsite of the command bar.
