Comment settings

The settings for customizing comment that are used for both comment-out/uncomment and comment highlighting are as follows:

inline commentThe string here is used as inline comment delimiter on comment-out/uncomment. If this field is empty, the values in block comment is used in favor.
block commentThe strings here are used as block comment delimiters. These delimiters are used on uncommenting block comments and comment-outing if no inline comment delimiter is defined. You need to fill both ”begin with” and ”end with”.

The coloring rules for the comment keywords in the table below are the same as the customizing syntax coloring rules. These settings are used only for highlighting.
