Customizing key bindings

You can fully customize CotEditor's menu key bindings. Also, you can store text snippets which have your preferred key bindings assigned. By default, you can insert “<br />” (with a line ending) by pressing Shift-Return. Go to “Key Binbings” pane in the preferences to create your own key bindings.

  • Menu Key Bindings must include Command key.
  • Snipett Key Bindings must exclude Command key.
  • You cannot set the key bindings that are already taken (e.g. shortcuts for your script files or menu items that don't appear in the Menu Key Bindings list).
