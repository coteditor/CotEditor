Getting information about character

CotEditor can popover the information about the selected character. Select exactly a single character and select “Inspect Character” in “Text” menu or press Cmd + Option + I key.

In the popover, zoomed glyph, its unicode character name, block name, general category, and its Unicode code point will be displayed. It shows also UTF-16 codes of each character in braces if the character is a surrogate pair. Only unicode codes are lined-up if the selected letter consists actually of multiple characters.
