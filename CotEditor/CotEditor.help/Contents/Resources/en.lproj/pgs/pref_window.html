Window preferences

The Window preferences allow you to set the default window appearance such as widow size or line number display.
Changes you make here will take effect when you next create/open a document.

Prefer Tabs
You can set whether documents are preferred to open as tabs. CotEditor respects your setting in the “Dock” system preferences if you set to “Respect System Setting.”
Show
You can set the visibility of parts in the window.
Line numbers
With this option turned on, CotEditor shows line numbers on the left side of the window.
Page guide
When this is turned on, a page guide (a vertical line) is displayed in the window's text editor area.
at column:
You can specify the column at which you want the page guide displayed. Note that the display position will differ according to your font setting.
The default font type can be changed in the Appearance pane.
Status bar shows
You can set the information to display in the status bar.
Window size
You can set the default window size in pixel.
Split editor
You can set the default orientation to split editors.
Writing direction
You can set the default writing direction in editors.
Overscroll
You can set the amount of the extra space at the bottom of the editor to scroll past end of a document. By 0%, the scroll stops after reaching the end of the document; and by 100%, the scroll continues until only the last line is visible.
Editor opacity
You can set the default editor opacity using the slider.
