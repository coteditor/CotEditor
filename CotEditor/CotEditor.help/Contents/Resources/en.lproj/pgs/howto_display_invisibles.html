Display invisible characters in CotEditor

The Invisible Characters feature allows you to draw invisible characters, such as special characters for formatting, in a document with alternative symbols so that you can find where they are.

Change visibility of invisible characters

Temporary changing the current document

You can switch the visibility of invisible characters in the current document by performing one of the following:

  • Choose Show/Hide Invisible Characters command from the Format menu.
  • Select the Invisibles button in the window toolbar (The Invisibles button can be added by customizing the toolbar).
  • Select the Invisibles button in the Touch Bar (The Invisibles button can be added by customizing the Touch Bar).

The change here doesn’t override the CotEditor’s default setting.

Default setting

You can change the default setting of the invisible visibility in the Appearance settings pane by selecting the “Show invisible character” checkbox.

Print invisible characters on printing

You can set whether printing also invisible characters on print in the Print settings pane for the default setting, or in the print dialog for an individual document.

Types of invisible characters

In the Appearance settings pane, you can set which types of invisible characters are shown when the invisible character display is enabled. The types correspond to the following characters:
