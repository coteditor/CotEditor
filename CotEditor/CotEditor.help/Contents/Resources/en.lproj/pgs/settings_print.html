Change Print settings in CotEditor

In the CotEditor app on your Mac, use Print settings to choose the default style for document printing. To change these settings, choose CotEditor > Settings, then click Print.

Open CotEditor for me

OptionDescription
Font The text font and font size for printing documents.
  • Same as document’s font: Use the same font used in the editor.
  • Select: Always use a specific font and font size for printing purpose. To change the font or font size, click Select, then choose a different font or font size.
Color The color setting for printing documents.
  • Black and White: Don’t print background and print all characters in black.
  • Same as document’s setting: Use the same theme used in the editor.
  • Theme name: Always use a specific theme for printing purpose.
Line numbers Print line numbers in front of each line.
Invisibles Print invisible characters in the document.
Header and Footer Choose the document information and their position to print in the header and footer of each page.
  • None: Print nothing.
  • Syntax Name: The name of the selected syntax style.
  • Document Name: The name of the document file.
  • File Path: The path to the document file.
  • Print Date: The current date and time.
  • Page Number: The number of each page.
