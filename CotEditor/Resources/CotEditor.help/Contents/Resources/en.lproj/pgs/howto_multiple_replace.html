Replace text by multiple rules in CotEditor on Mac

With the Multiple Replace feature, you can apply multiple text replacements at once in sequence. Each set of replacement rules can be saved as a named preset and reused whenever needed.

Use Multiple Replace window

Use the Multiple Replace window to manage and perform the multiple replacements in your document.

To open the Multiple Replace window:

  1. Go to the CotEditor app on your Mac.

  2. Choose Find > Multiple Replace.

Manage replacement definitions

The stored replacement definitions you have created before appear in the list on the sidebar.

  • Click the Add button (+) at the bottom of the list to create a new definition.

  • Click the Delete button (-) at the bottom of the list to delete the selected definition.

  • Click the selected definition in the list again to rename it.

  • Click the Action pop-up menu () for further operations.

Export/import replacement definitions

You can export replacement definitions to transfer them to another computer. Each definition is saved as a CotEditor replace definition file with the .cotrpl extension. The file content is written in JSON format.

  • Export: Select a definition in the list, then choose Export or Share from the Action menu.

  • Import: Drop replacement definition files files into the list, or choose Import from the Action menu, then select the files to import.

Import TSV data as replacement definitions

You can also create a replace definition from a UTF-8–encoded TSV file in which each line contains a pair of search and replacement texts separated by tabs. To import this format, choose a file with the .tsv extension using the Import command. In this format, you can’t specify options such as case sensitivity or enabling regular expressions. After importing a TSV file as a replace definition, configure any necessary options manually.

Perform multiple replacements

The definition selected in the list appears in the editor area on the right side. The replacement rules in the editor area are performed row by row from top to bottom at once.

Click the buttons below the editor area to perform multiple replacements.

  • Replace All: Perform all replacements for the current document.

  • Highlight: Highlight all matches in the current document*.

* The matches highlighted by the Highlight command may differ from the actual replacement results because Replace All applies rules sequentially, and each rule is applied to the document after previous rules have already taken effect.

Edit replacement definition

You can create and customize your own replacement definitions.

Create a new replacement definition

  1. Click the Add button (+) at the bottom of the list on the left side to create a new replacement definition.

  2. Click the Add button (+) in the editor area on the right side to create a new replacement rule.

  3. Enter your replacement rule in the added row.

Your changes are saved automatically as you edit.

If a row contains an invalid rule, it is skipped when performing the replacement.

Replacement rule options

OptionDescription
(checkbox)If the checkbox is unselected, the row is just skipped. You can use this option to keep a replacement rule that is used depending on the situation.
FindThe text to search for. This field cannot be left empty. If empty, the rule is considered invalid and skipped.
Replace WithThe text to replace with. If left empty, the matched text is removed.
RESearch using a regular expression (abbreviation of Regular Expression).
ICIgnore case when searching (abbreviation of Ignore Case).
DescriptionYou can use this field to leave a comment about the rule. The description doesn’t affect to replacement. This field can be left blank.

Change advanced find options

You can fine-tune the find behavior for the entire replacement definition in the Advanced options view. Those settings are also saved to your definition and apply to all rules within the definition.

Click the gear button in the Multiple Replace window, then click checkboxes as needed.

For details on each setting, see Find or replace text.
