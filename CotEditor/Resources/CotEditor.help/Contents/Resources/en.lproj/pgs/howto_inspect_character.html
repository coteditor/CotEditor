Get information about a character in CotEditor on Mac

CotEditor can inspect the information about the selected character.

  1. In the CotEditor app on your Mac, select exactly a single character.

  2. Choose Text > Inspect Character.

The inspector shows: the character in close-up, the Unicode code point, the Unicode character name, block name, general category, and the Unicode version that introduced the character. It also shows the UTF-16 codepoints of each character in braces if the character is a surrogate pair. If the selected letter consists actually of multiple Unicode characters, only the code points and Unicode names appear.

Clicking on a block name in the inspector switches between the localized block name and the block name defined in Unicode.
