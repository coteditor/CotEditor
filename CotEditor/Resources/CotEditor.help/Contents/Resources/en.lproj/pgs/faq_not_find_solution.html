If you can’t find a solution in CotEditor on Mac

If you still can’t find a solution, please contact the development team. Choose Help > Create Issue Report to create a report template with your environment information already filled in.

Send the completed report in one of the following ways:

  • Post it on GitHub Issues (recommended)

  • Send it by email to coteditor.github@gmail.com

Please write your message in either English or Japanese.
