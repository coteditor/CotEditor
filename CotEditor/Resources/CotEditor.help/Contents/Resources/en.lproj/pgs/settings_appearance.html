Change Appearance settings in CotEditor on Mac

Use Appearance settings in CotEditor to set how documents appear, including font, line height, and colors.

To change these settings in the CotEditor app on your Mac, choose CotEditor > Settings, then click Appearance.

OptionDescription
Standard font / Monospaced font Set the standard and monospaced fonts used in editors. Set which font is used for which document in Mode settings.
Font
The font and font size. To change the font, click the Select button, then choose a font name or font size in the Fonts window.
Antialias
Apply antialiasing to text. Antialias is a rendering feature that smoothes jagged edges.
Ligatures
Apply the standard ligature rules to the editor’s text, or turn off all ligatures. A ligature is a typographic feature joining specific characters as a single glyph.
Line height Set the line height as a multiple of the default line height (from 0.1 to 10.0).
Appearance Choose the document appearance independently of the system appearance.
Editor opacity Drag the slider or enter a percentage to change the editor’s opacity.
Theme Choose your default color theme for editors and manage themes. To learn more about customization, see Customize themes.
Theme list Choose the default theme for editors.
Add +, Delete - Create a new theme or delete the selected theme. Built-in themes can’t be removed.
Action pop-up menu
Duplicate
Duplicate the selected theme to create your own version.
Restore
Restore the selected theme to its default state. Available only for built-in themes.
Export
Export a theme file created from the selected theme. With exported files, custom themes can be used on another computer. To use an exported file, choose Import.
Share
Share a theme file created from the selected theme.
Import
Import a theme created on another computer. After choosing Import, select the theme file you want to add.
Theme editor Edit the selected theme. Click the color wells to choose colors.
