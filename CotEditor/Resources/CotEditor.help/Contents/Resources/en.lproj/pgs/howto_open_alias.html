Open an alias or a symlink in CotEditor on Mac

You can open the original file or folder at the target of an alias or a symlink as a document.

Because CotEditor is sandboxed, it can’t open documents or folders outside the sandbox directly from the file browser in the sidebar. Therefore, CotEditor may show an Open dialog to request access when you try to open the original item. Once permission is granted, CotEditor can open the original item again without showing the dialog until you quit the app.

Open the original document or folder directly

When you open an alias or a symlink in the CotEditor app on your Mac, the app opens its original file or folder just like opening a regular document.

Open an original document from the file browser

  1. In the CotEditor app on your Mac, open a folder document.

  2. Select an alias or a symlink to a file in the file browser on the sidebar.

    If permission hasn’t been granted yet, or if the original item doesn’t appear to be a text file, a preview is shown in the document pane.

  3. Click Open Original.

    The Open dialog appears.

  4. Click Open.

Open an original folder from the file browser

  1. In the CotEditor app on your Mac, open a folder document.

  2. Select an alias or a symlink to a folder in the file browser on the sidebar.

    A preview is shown in the document pane.

  3. Click Open in New Window.

    If permission hasn’t been granted yet, the Open dialog appears.

  4. Click Open.

Note: CotEditor can’t expand the original folder of an alias or a symlink directly inside the file browser. Instead, it opens the original folder in a new window.
