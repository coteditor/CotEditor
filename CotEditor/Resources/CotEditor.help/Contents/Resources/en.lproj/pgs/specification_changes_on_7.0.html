Important changes on CotEditor 7.0

In CotEditor 7.0.0, released in April 2026, the syntax definition specifications have been revised. Therefore, users who create or distribute custom syntaxes may need to adjust their custom syntaxes.

For the latest CotEditor syntax definition specifications, see Define a syntax.
