Script CotEditor on Mac

CotEditor includes a scripting feature that lets you run scripts in CotEditor to process documents or launch external tools. Your scripts appear in the Script menu represented with .

Scripting types

On CotEditor, there are two different ways to write scripts:

AppleScript
AppleScript (incl. JXA; JavaScript for Automation) can control documents in CotEditor with a high level of flexibility.
UNIX script
UNIX scripts can modify only the frontmost document, but you can write them in various languages such as Python, Ruby, and Swift.

Choose the scripting type that suits your needs. For details about how to write your scripts, see Script CotEditor with AppleScript or Script CotEditor with UNIX script.

Add or delete scripts

To add scripts to or remove scripts from the Script menu, follow these steps:

  1. Choose Script > Open Scripts Folder in CotEditor. The Finder opens the folder for CotEditor scripts at:
    ~/Library/Application Scripts/com.coteditor.CotEditor/
  2. Put your script files in this folder, or remove unwanted ones.
  3. Changes to this folder are applied automatically to the Script menu.

You can also change the order of scripts or assign keyboard shortcuts by modifying the names of your script files. See Customize the Script menu for the details.

Get existing CotEditor scripts

The CotEditor project provides sample scripts. You can download them from:
