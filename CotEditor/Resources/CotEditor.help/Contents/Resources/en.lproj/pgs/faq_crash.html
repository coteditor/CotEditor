If CotEditor on Mac quits unexpectedly

If CotEditor quits unexpectedly again and again under specific conditions, please report it with the crash log attached.

  • GitHub Issues

When you send the crash log, include the following information as well:

  • The operation you performed just before the unexpected quit.

  • Your environment information, such as the macOS version and the CotEditor version.

You can view the crash log from the dialog that appears immediately after the crash.
