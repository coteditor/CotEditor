Customize the syntax menus in CotEditor on Mac

CotEditor includes syntax definitions for many languages and file formats. You can remove syntaxes you don’t usually use from the syntax menus so that the syntaxes you use often are easier to find.

Hiding a syntax removes it from the Syntax menus in the toolbar and the Format menu. Hidden syntaxes are still used for automatic syntax detection when you open a document or rename a file.

Choose which syntaxes appear in the syntax menus

  1. Go to the CotEditor app on your Mac.

  2. Choose CotEditor > Settings, then click Format.

  3. In the “Available syntaxes” section, click at the bottom of the list, then choose Customize Syntax Menu.

  4. Select or deselect each syntax’s checkbox to show or hide it in syntax menus.

  5. To select every syntax in the list, click Select All.

  6. Click OK.
