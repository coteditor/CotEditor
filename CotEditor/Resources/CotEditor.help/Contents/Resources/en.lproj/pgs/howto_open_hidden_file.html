Open hidden files in CotEditor on Mac

The checkbox “Show hidden files” in the Open dialog can switch the visibility of hidden files in the dialog.

Show hidden files in the file browser

  1. Go to the CotEditor app on your Mac.

  2. Open the context menu in the file browser on the sidebar.

  3. Choose Show Hidden Files.

To hide hidden files again, select Show Hidden Files once more and remove the check mark.
