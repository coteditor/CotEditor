Print a document in CotEditor on Mac

Use the Print dialog to print documents or export them as PDF files. You can also customize print settings to change the print range, included information, and appearance.

Print a document

  1. In the CotEditor app on your Mac, open a document.

  2. To print only part of a document, select the text you want to print.

  3. Choose File > Print.

    The Print dialog appears.

  4. Choose printing options.

  5. Click Print.

You can change your print settings how the document will be printed in the CotEditor section in the Print dialog. The settings can be saved with other general print settings as a preset in the Print dialog.

OptionDescription
Font size

The text size for printing documents.
Color

The color setting for printing documents.
Line numbers

Print line numbers in front of each line.
Invisibles

Print invisible characters in the document.
Header and Footer

Select the document information and their position to print in the header and footer of each page.

  • None: Print nothing.
  • Syntax Name: The name of the selected syntax.
  • Document Name: The name of the document file.
  • File Path: The path to the document file.
  • Print Date: The current date and time.
  • Last Modified Date: The date and time when the document was lastly modified.
  • Page Number: The number of each page.

Tip: See Print documents from your Mac for other general print options.
