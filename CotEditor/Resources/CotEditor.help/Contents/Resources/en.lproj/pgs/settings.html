Change settings in CotEditor on Mac

Use CotEditor settings to make the editorâ€™s appearance, behavior, and document handling work best for you.

  1. Go to the CotEditor app on your Mac.

  2. Choose CotEditor > Settings, then click a settings pane:

    • General: Choose how documents are opened and saved and to adjust other basic behaviors.

    • Appearance: Set how documents appear, including font, line height, and colors.

    • Window: Control how document windows look and behave.

    • Edit: Choose how the editor behaves while editing.

    • Mode: Adjust how editors behave depending on the document type.

    • Format: Set the default options for documents, such as line endings and text encoding, and to manage syntaxes.

    • Snippets: Define preset text to create preset text that you can insert with a command or by dropping a file onto the editor.

    • Key Bindings: Change keyboard shortcuts for menu commands.

    • Donation: Support the CotEditor project by offering coffee.

To learn more about the options in a pane, click at the bottom of the pane.
