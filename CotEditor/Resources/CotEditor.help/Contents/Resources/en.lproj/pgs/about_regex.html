Use the regular expression in CotEditor on Mac

In CotEditor, you can use regular expression patterns in features such as Find and Replace and line sorting to define more flexible matching conditions. Regular expressions are also used to specify rules for syntax highlighting and outline extraction.

CotEditor uses an ICU-compatible library for regular expressions. For details about the regular expression syntax, see the ICU Regular Expressions Guide.

You can also customize detailed behaviors when performing regular expression searches and replacements. For more information about customization options, see Find and replace text.
