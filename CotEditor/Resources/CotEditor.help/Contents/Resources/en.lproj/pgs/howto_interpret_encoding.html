Open a document with a specified encoding in CotEditor on Mac

You can open a document by interpreting it with a specified text encoding. This is useful when a document doesn’t appear correctly or appears garbled.

Specify the encoding when opening a document

  1. Go to the CotEditor app on your Mac.
  2. Choose File > Open.
  3. Select the document you want to open.
  4. From the Text Encoding pop-up menu, select the encoding you want to use.
  5. Click Open.

The default setting is Automatic. When Automatic is selected, CotEditor examines the document’s content and the text encoding priority list, then automatically chooses the most appropriate encoding. For more information about encoding priorities, see Change the priority of text encodings.

If you open a document by dragging it to the CotEditor application icon, CotEditor uses an automatically detected encoding. If you need to open a document with a specific encoding, use the Open dialog.

Reinterpret a document using a different encoding

If a document doesn’t appear correctly, you may be able to fix the issue by reinterpreting it using a different encoding.

To reinterpret an open document using another encoding, do the following:

  1. Go to the CotEditor app on your Mac.
  2. Click the Text Encoding button in the status bar, or choose Format > Text Encoding.
  3. From the list, select the encoding you want to use.
  4. In the “Text encoding change” dialog that appears, click Reinterpret.

When reinterpreting a document using a different encoding, note that any unsaved changes are lost. To preserve the content, save a copy beforehand.

If you don’t see the encoding you want in the Text Encoding menu

If you don’t see the encoding you want in the Text Encoding menu, you may be able to customize the list and add the encoding you need.

The encodings that appear in the Text Encoding menu can be changed in Format settings under “Encoding priorities.” For more information, see Change the priority of text encodings.
