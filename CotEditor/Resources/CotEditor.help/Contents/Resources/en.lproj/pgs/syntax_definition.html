Define a syntax in CotEditor on Mac

Use the syntax editor in CotEditor to edit the built-in syntaxes or create your own.

Open the syntax editor

To open the syntax editor, do the following:

  1. Go to the CotEditor app on your Mac.

  2. Choose CotEditor > Settings, then click Format.

    • To customize an existing syntax:

      1. Select the syntax you want to customize in the “available syntaxes” list.

      2. Double-click the syntax name, or click the Edit button.

    • To define a new syntax:

      1. Click the Add button below the “available syntaxes” list.

Edit a syntax definition

In the syntax editor, you can configure the following features:

  • Name: Set the name of the syntax.

  • Kind: Specify whether the syntax is for coding or for general purpose. This option affects the editor’s font and behavior set in Mode settings.

  • highlighting elements: Define the rules for syntax highlighting.

  • Comments: Define comment delimiters and related highlighting rules.

  • Outline: Define how outline items are extracted and formatted.

  • Completion: Define completion candidates.

  • File Mapping: Define what kind of files use this syntax.

  • Information: Edit metadata for the syntax definition.

  • Validation: Verify that no grammatical or formatting errors exist in the definition.
