Export/Import settings in CotEditor on Mac

Export CotEditor settings to an archive file, then import them into CotEditor on another computer.

The settings archive can includes:

ItemDescription
Settings All options that can be configured within the Settings window, such as modes, snippets, and key bindings
Themes All custom themes defined by the user
Syntaxes All custom syntaxes defined by the user.
Multiple Replace definitions All definitions set in the Find > Multiple Replace menu.

The exported data is saved as a CotEditor Settings Archive (.cotsettings) file.

You can also export themes, syntaxes, and Multiple Replace definitions individually. For details, see Change Appearance settings,” “Change Format settings, and Replace text by multiple rules.

Export settings

To export your settings in CotEditor:

  1. Choose File > Export Settings.
  2. Select the items to export.
    • All exportable items are automatically extracted.
  3. Click Export.
  4. Enter a name and choose a location for the exported archive, then click Save.

Import settings

To import your settings in CotEditor

  1. Choose File > Import Settings.
  2. Choose the CotEditor settings archive to import.
  3. Select the specific items in the archive you want to import.
  4. Click Import.
    • If CotEditor already has a setting with the same name, it will be overwritten with the imported data.
